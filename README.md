# PulseStrategy Frontend

A professional front-end interface for the PulseStrategy smart contract on Ethereum Mainnet.

## Deployment
1. Push this repository to GitHub.
2. Connect to a hosting platform:
   - **Vercel**: Import repo, set Framework to Vite, Build Command to `npm run build`, Output Directory to `dist`.
   - **Netlify**: Import repo, set Build Command to `npm run build`, Publish Directory to `dist`.
   - **GitHub Pages**: Enable Pages in repo settings, set source to `gh-pages` branch or use `dist` folder.
   - **Cloudflare Pages**: Import repo, set Build Command to `npm run build`, Output Directory to `dist`.
